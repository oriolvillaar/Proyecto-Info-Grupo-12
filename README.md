VERSION 1:
En esta versión, el software procesa una lista de aeropuertos y permite modificarla ya sea añadiendo o eliminando aeropuertos de dicha lista, procesando a la vez un fichero KML para su visualización en Google Earth con un código de colores que permite distingir entre aeropuertos pertenecientes o no a la zona Schengen (verde en el caso de que pertenezcan y rojo en el que no). La interfaz nos permite ver las coordenadas de cada aeropuerto y si pertenecen o no a la zona Schengen.

Version 2: en esta versión añadimos las funciones para el trabajo con el fichero arrivals, a partir del cual establecemos gráficos sobre aerolíneas y tipos de vuelo operando desde LEBL y añadiendo también otro fichero KML para su visualizacion en Google Earth sobre las rutas operadad con origen Barcelona. También filtramos el tipo de vuelo según duración y los guardamls en un fichero nuevo. LINK VIDEO DRIVE: https://drive.google.com/file/d/1jmBpO5TqDN1XXCth1NTdhy3Z0pjCSNdJ/view?usp=drivesdk


Version 3: en esta versión añadimos al software los datos sobre la disposición del aeropuerto con sus diferentes terminales y zonas de embarque y leemos los datos sobre la disposición de puertas y zonas segun tipo de vuelo y aerolínea para después poder proceder a la asignación de puertas a cada vuelo. Además, elaboramos un gráfico representativo de las diferentes zonas que, junto con un código de color, muestran la ocupación de cada gate del aeropuerto.
